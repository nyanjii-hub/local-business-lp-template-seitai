# 整体院向けLPテンプレート

地域密着の整体院向けランディングページテンプレートです。肩こり・腰痛・姿勢改善などの悩みを訴求し、LINE予約・相談、電話予約、Instagram、店舗情報までを1ページで案内できます。

## 特徴

- スマートフォン優先のレスポンシブデザイン
- LINE予約・相談、電話予約、Instagram導線
- 画面下部の固定CTA
- 施術メニュー、患者様の声、施術風景、店舗情報、Google Mapセクション
- HTML/CSS/JavaScriptのみで動作
- Vercel / Netlify / GitHub Pages にそのまま公開可能

## 使い方

1. `index.html` を開く
2. 以下の項目を実際の整体院情報に差し替える
3. GitHubにアップロードする
4. Vercel / Netlify / GitHub Pages で公開する

## 変更する主な項目

| 項目 | 検索する文字 | 変更例 |
|---|---|---|
| 院名 | `Your Seitai` / `Your Shop` | `〇〇整体院` |
| 電話番号 | `000-0000-0000` / `0000000000` | `03-1234-5678` / `0312345678` |
| LINE | `@your_line_id` | `@sample_seitai` |
| Instagram | `your_account` | `sample_seitai` |
| 住所 | `東京都中央区銀座1-2-3` | 実際の住所 |
| 施術メニュー | `肩こり改善整体` など | 自院の施術メニュー |
| 料金 | `¥4,800` など | 自院の料金 |
| 営業時間 | `平日 10:00〜19:00` など | 自院の営業時間 |
| 患者様の声 | サンプル文 | 実際の声や掲載許可済みの内容 |

## Google Mapについて

このテンプレートではAPIキー不要の埋め込みURLを使っています。

```html
https://www.google.com/maps?q=東京都中央区銀座1-2-3&output=embed
```

住所部分を整体院の住所に変更してください。道案内ボタン側のURLも同じ住所へ差し替えると自然です。

## 注意

- 画像はUnsplashの外部画像を使用しています。商用利用時は必要に応じて自院の写真に差し替えてください。
- 患者様の声はサンプルです。実際の声を掲載する場合は、掲載許可や引用ルールを確認してください。
- 電話番号、LINE、Instagram、住所、営業時間は公開前に必ず差し替えてください。
- 医療的な効果を保証する表現は避け、必要に応じて専門家や広告ガイドラインを確認してください。
